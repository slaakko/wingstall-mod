Footnote 1. %WINGSTALL_ROOT% represents an environment variable whose name is WINGSTALL_ROOT and value is a path to the Wingstall installation directory. By default the installation directory is 'C:\wingstall'. You can check the value of the WINGSTALL_ROOT environment variable from Windows Control Panel | System | Advanced system settings | Environment Variables. You can also open the Windows Explorer, paste the path that contains %WINGSTALL_ROOT% to the address bar and press enter.
