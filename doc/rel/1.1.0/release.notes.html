Wingstall 1.1.0 for Windows x64

Release notes

  • Bug fix: Progress percentages fixed in wingstall --create-package command, install window and uninstall window.

  • Bug fix: For some packages the bzip2 library reported a parameter error when compressing. This is probably a bug in my BZip2Stream class (note: this is not a bug in the bzip2 library, I think). Resolved by setting the buffer size of the BufferedStream the same as the buffer size of the BZip2Stream (16K).

  • Error reporting of the XML parser improved. Now the XML parser reports error line and column in case of a parsing error.

  • Source code for the example project included.

  • Documentation revised.
