Wingstall 2.1.0 for Windows x64

Release notes

  • Wingstall versions 1.0.0 and 1.1.0 did not work at all so I decided to remove them from SourceForge. Their initialization code depended on an environment variable named WINGSTALL_ROOT being defined in the machine. If that was not the case the installer crashed. This is now fixed. Apologies to those who downloaded it and tried to use it.

  • The package editor does not support creation of preinstall and uninstall components but they are now carried in the package XML if they are defined.

  • Bug fix: fixed a bug in Path::ChangeExtension function.
